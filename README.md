1 - Project Description – EventsX

EventsX is a dynamic and responsive web application developed as part of the CSE211 Web Programming course project. The project simulates an online event planner platform that allows users to browse upcoming events, register for events, calculate event-related budgets, and interact with event data dynamically on the client side.

The website is built using HTML5 semantic elements, CSS3 with Flexbox, and object-oriented JavaScript (ES6) to ensure clean structure, responsiveness, and interactive user experience. All functionalities are implemented on the client side, focusing on DOM manipulation, form validation, and dynamic content rendering using JavaScript.

2 - Project Objectives :

Design a responsive multi-page event planner website.

Apply HTML5 semantic structure and validation standards.

Use CSS3 Flexbox for layout and responsive design.

Implement interactive features using JavaScript ES6 classes.

Validate user input using client-side JavaScript.

Dynamically display and filter event data without page reloads.

3 - Technologies Used

HTML5 – Semantic page structure

CSS3 – Flexbox, transitions, animations, responsive layout

JavaScript (ES6) – DOM manipulation, classes, custom events

4 - Main Features

Display upcoming events dynamically.

Search and filter events by keyword.

Event registration form with client-side validation.

Budget calculator for event-related costs.

Responsive sliding registration form.

Parallax scrolling effect.

Fully responsive design for desktop, tablet, and mobile devices.

5 - Project Structure

The project follows a logical and organized folder structure as required by the course specifications, separating HTML pages, CSS stylesheets, JavaScript files, and media assets to ensure maintainability and clarity.

6 - Notes

All pages are implemented using valid HTML5 semantic elements.

The website is responsive and compatible with modern web browsers.

All interactive functionality is implemented using raw JavaScript without any external frameworks.

AI tools were used for learning and assistance and are documented as required by the course guidelines.
